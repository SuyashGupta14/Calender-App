# 📅 Wall Calendar

A modern, interactive wall calendar application built with **Next.js 16**, **React 19**, and **TypeScript**. Features a premium dual-theme design, date range selection, persistent notes, holiday indicators, and smooth directional animations.

## ✨ Features

| Feature | Description |
|---------|-------------|
| 🎨 **Dual Theme** | Premium light & dark themes with smooth transitions via CSS custom properties |
| 📆 **Date Range Selection** | Click a start date, then an end date to highlight a range |
| 📝 **Persistent Notes** | Add, edit, and delete notes for any date range — saved in `localStorage` |
| 🖼️ **Hero Images** | Unique, high-quality Unsplash images for each month with directional slide animations |
| 🎌 **Indian Holidays** | Auto-highlights Indian public holidays with red dot indicators |
| 🎬 **Smooth Animations** | Direction-aware slide transitions, staggered day cell animations, and fade effects |
| 📱 **Responsive Design** | Adapts seamlessly from desktop to mobile viewports |
| ⚡ **Zero Dependencies** | No extra libraries — just Next.js, React, and TypeScript |


## Getting Started

### Prerequisites

- **Node.js** 18.17 or later — [Download](https://nodejs.org/)
- **npm** (comes with Node.js) or **yarn** / **pnpm**

### Clone & Install

```bash
# Clone the repository
git clone https://github.com/SuyashGupta14/Calender-App.git

# Navigate to the project
cd Calender-App

# Install dependencies
npm install
```

### Run Development Server

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

### Build for Production

```bash
npm run build
npm start
```
